📱 HOW IT WORKS NOW:

1. SITTER sees booking:
   ┌─────────────────────────┐
   │ Booking with Ali        │
   │ Status: Requested       │
   │                         │
   │ [Accept]  [Decline]     │ ← For Sitter
   └─────────────────────────┘

2. SITTER clicks Accept:
   → Status changes to "Confirmed"
   → Owner gets notification

3. OWNER sees booking:
   ┌─────────────────────────┐
   │ Booking with Ali        │
   │ Status: Confirmed       │
   │                         │
   │ [💳 Pay Now - €50.00]   │ ← NEW! For Owner
   └─────────────────────────┘

4. OWNER clicks Pay Now:
   → Redirects to Stripe payment page
   → Shows: Total €50 (€40 to sitter, €10 to you)
   → Owner enters card details
   → Payment held in escrow

5. After Payment:
   ┌─────────────────────────┐
   │ Booking with Ali        │
   │ Status: Confirmed       │
   │                         │
   │ ✓ Payment secured -     │
   │   Waiting for service   │
   └─────────────────────────┘

6. Service Completed:
   → Owner clicks "Release Payment"
   → €40 goes to sitter
   → €10 goes to you (platform)
   → Done! 🎉

═══════════════════════════════════════════════════════════════

💡 BUTTON RULES:

WHO SEES WHAT:

SITTER sees:
  • [Accept] [Decline] → on "requested" bookings

OWNER sees:
  • [💳 Pay Now] → on "confirmed" bookings (no payment yet)
  • [Release Payment] → on "completed" bookings (payment held)
  • [Cancel & Refund] → on "requested" bookings (payment held)
